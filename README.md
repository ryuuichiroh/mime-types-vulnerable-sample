# MIME Types Vulnerability Sample

このプロジェクトは、OSSの脆弱性検出を検証するためのサンプルプロジェクトです。

## 概要

`mime-types` パッケージのバージョン 2.1.18 を使用しています。
このバージョンには既知の脆弱性が存在する可能性があり、脆弱性スキャンツールによる検出を検証することができます。

## セットアップ

```bash
# 依存関係のインストール
npm install

# アプリケーションの実行
npm start
```

## 使用しているパッケージ

- `mime-types@2.1.18` - MIMEタイプの検索とコンテンツタイプの生成

## 脆弱性スキャン

このプロジェクトで脆弱性スキャンツールをテストできます：

```bash
# npm audit で脆弱性をチェック
npm audit

# Snyk などの他のツールでもスキャン可能
```

## 注意

このプロジェクトは脆弱性検出のテスト目的のみで作成されています。本番環境では使用しないでください。
